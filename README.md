# Open Redirect PoC - CVE-2025-58044

## 📋 Description

This is a Proof of Concept (PoC) for **CVE-2025-58044**, an Open Redirect vulnerability in JumpServer that allows attackers to redirect users to arbitrary websites by manipulating the `Referer` header.

**Vulnerability:** Open Redirect via Referer Header  
**Affected Product:** JumpServer < v3.10.19 and < v4.10.5  
**CVSS Score:** 6.1 (Medium)  
**CVE:** CVE-2025-58044
