📋 How It Works

This vulnerability allows an attacker to redirect users to any malicious website by manipulating the Referer header.

1 Attacker sets Referer header to malicious URL
2 Target server redirects to the Referer URL
3 Victim is redirected to attacker-controlled phishing page
